# /timing SYMBOL — 择时分析

**目的**：综合技术信号共振、IV 环境、估值分位，给出明确的入场 / 离场 / 等待建议。

## 执行步骤

### Step 1：技术信号详情

```bash
cd D:\AI_about\invest_analysis
python tui_query.py analyze SYMBOL
```

重点关注：
- 信号共振程度（多少个指标同向）
- 当前价格与支撑/压力位的关系
- 止损止盈参考位

### Step 2：IV 环境

```bash
python tui_query.py iv SYMBOL
```

关注：
- 当前 IV vs 历史 IV 百分位
- IV 环境对期权策略的影响（IV 高 → 卖方有利；IV 低 → 买方有利）

### Step 3：估值快照

```bash
python tui_query.py valuation SYMBOL
```

只需提取：PE 百分位（当前贵不贵）

### Step 4：输出择时判断

```
# SYMBOL 择时分析
日期：[今天日期]  |  当前价：[从 analyze 输出读取]

## 信号共振
- [指标1]：[状态]
- [指标2]：[状态]
- [指标3]：[状态]
共振强度：[X/5 个指标同向]

## 技术关键位
- 支撑位：[值]（来自止损参考）
- 压力位：[值]（来自止盈参考）
- 当前位置：[价格在支撑/压力之间的位置描述]

## IV 环境
- 当前 IV：[值]  |  历史百分位：[X%]
- 含义：[IV 偏高/偏低，对应适合的策略风格]

## 估值位置
- PE 百分位：[X%]（[便宜/合理/贵]）

## 择时结论

**建议**：[买入 / 等待 / 减仓 / 不操作]

理由：
[2-3 条具体依据，直接引用上面的数据]

**注意**：
[止损位置，或者什么情况下结论失效]
```

## 判断框架

- **强买入时机**：信号共振 4/5+，IV 中性偏低，PE 百分位 < 50%
- **谨慎买入**：信号共振 3/5，但 IV 偏高或 PE 偏贵
- **等待**：信号混乱（< 3/5 同向），或技术位于压力位附近
- **减仓信号**：信号转空头 + PE 高于 p80 + 止盈位触及

## 注意事项

- 择时不是预测，是概率判断
- 信号共振不足时，明确说"当前信号不清晰，建议等待"
- IAU、BND、QQQ 无择时意义，直接告知
