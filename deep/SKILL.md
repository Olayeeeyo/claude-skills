# /deep SYMBOL — 深度分析

**目的**：整合技术信号、估值数据、财报与分析师预期，给出完整的投研报告。

## 执行步骤

### Step 1：综合技术 + 估值分析

```bash
cd D:\AI_about\invest_analysis
python tui_query.py analyze SYMBOL
```

提取：综合评分、信号强度、各技术指标状态、止损止盈参考位

### Step 2：估值详情

```bash
python tui_query.py valuation SYMBOL
```

提取：当前 PE、历史 PE 百分位（p20/p50/p80）、forward EPS 趋势、SA 评级

### Step 3：财报 & 分析师预期

```bash
python tui_query.py earnings SYMBOL
```

提取：最近一季 EPS（实际 vs 预期）、营收预期、分析师评级分布

### Step 4：输出完整报告

```
# SYMBOL 深度分析报告
日期：[今天日期]

## 1. 综合评分
总分：X/100  |  信号：[买入/中性/卖出]
强度：[弱/中/强]

## 2. 技术面
- 趋势：[MA 排列状态]
- 动量：MACD [多头/空头背离/金叉/死叉]，RSI [值]（[超买/超卖/中性]）
- 布林带：[收窄/扩张，当前价格位置]
- 止损参考：[来自 analyze 输出]
- 止盈参考：[来自 analyze 输出]

## 3. 估值面
- 当前 PE：[值]  |  历史百分位：[X%]（[偏低/合理/偏高]）
- Forward EPS：[值]  vs  Trailing EPS：[值]（[增长/下降 X%]）
- PE 区间参考：p20=[值] / p50=[值] / p80=[值]
- SA 量化评级：[值]  |  分析师评级：[买入X / 持有Y / 卖出Z]

## 4. 基本面
- 最近财报：EPS [实际] vs 预期 [预期]（[超预期/不及预期 X%]）
- 营收：[实际] vs 预期（[超/不及]）
- 下次财报：[日期或"未知"]

## 5. 综合结论
[2-3 句话：当前是否是好的买入机会，主要机会和风险各是什么，建议如何操作]

是否需要做择时分析？→ /timing SYMBOL
```

## 注意事项

- IAU、BND、QQQ 无需做深度分析，直接告知用户这些是长期持有 ETF
- PE 百分位数据不足 12 个月时，估值分位仅供参考
- 技术信号和基本面同向时，结论置信度更高
